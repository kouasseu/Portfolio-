WAM SSO – Refonte de la plateforme d’authentification centralisée avec monitoring SafeKit

Description
Refondre la plateforme SSO (Single Sign-On) basée sur Evidian Web Access Manager (WAM) pour Orange Cameroun, afin de centraliser et sécuriser l’authentification à l’ensemble des applications métiers (CRM, Vue360). Le projet intègre ADFS comme fournisseur d’identité (IdP) via SAML v2, et s’appuie sur le module SafeKit pour assurer la haute disponibilité, la réplication et le monitoring du cluster WAM

Mon rôle

Ingénieur infra IT :

  • Analyse de l’existant et définition de l’architecture cible SSO
  • Déploiement et configuration de la plateforme Evidian WAM
  • Mise en œuvre de l’intégration SAML v2 avec ADFS comme IdP
  • Intégration des applications CRM, portail Evidian et Vue360 (Microsoft) dans le périmètre SSO
  • Déploiement du module SafeKit pour :
    • Réplication des données critiques entre nœuds
    • Supervision de l’état du cluster WAM
    • Bascule automatique en cas de défaillance
  • Coordination des équipes techniques locales et des éditeurs
  • Rédaction de la documentation technique et des guides d’intégration

Stack technique

  • Evidian Web Access Manager (WAM) : solution SSO centralisée
  • ADFS (Active Directory Fédération Services) : fournisseur d’identité (IdP)
  • SAML v2 : fédération d’identités et authentification inter-applications
  • SafeKit Evidian : réplication, supervision et bascule du cluster WAM
  • Microsoft Vue360 : application métier intégrée au SSO
  • LDAP / Active Directory : sources d’identité
  • HTTPS / Reverse Proxy : sécurisation des flux d’authentification
  • Redhat Linux : Serveurs hébergeant l’application WAM SSO
  • Windows Server 2016 : Serveurs fournissant le service ADFS

Résultats obtenus

  • Mise en place d’une authentification unifiée pour les applications CRM et portail
  • Renforcement de la sécurité via fédération SAML et centralisation des accès
  • Haute disponibilité assurée par le module SafeKit avec réplication et supervision
  • Réduction du RTO grâce à la bascule automatique en cas de panne
